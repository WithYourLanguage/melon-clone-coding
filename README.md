# Melon Contest Blueprint

## How to run?

- Open this folder on VSCode.
- Click on `View` -> `Terminal`.
- Run `npm i`.
- Start `mongodb`.
- Change the `COOKIE_SECRET` on the `.env` file.
- Run `npm run dev:server`.
- Open another terminal and run `npm run dev:assets`.

## How to deploy?

- Run `npm run build`.
- Run `npm start`.
